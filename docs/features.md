# 機能仕様

## 制作目的

金沢で特に有名な魚種（例：かに・のどぐろ・エビ）以外の消費者の認知度向上

## 機能一覧

### 食べ物系

| 機能名 | 説明 |
|--------|------|
| レシピ提案 | 魚種に応じたレシピを提案 |
| 食べた記録（日記） | 食べた魚の写真を記録 |
| 魚に合う日本酒 | その魚に合う日本酒を提案 |
| 魚のきれいな食べ方 | 魚の食べ方ガイド |
| 栄養成分比較ツール | 魚種間の栄養成分を比較 |
| 図鑑 | 魚のおすすめ機能（有名な魚種以外も） |

### 旅行系

| 機能名 | 説明 |
|--------|------|
| おすすめ店舗 | 魚料理が美味しい店舗を紹介 |
| 漁港ツアー | 漁港見学ツアー情報 |
| 食べ歩きスポットの提案 | 地域の食べ歩きスポットを提案 |

### 娯楽系

| 機能名 | 説明 |
|--------|------|
| 魚クイズ | 魚に関するクイズゲーム |
| 魚の一生体験ミニゲーム | 自分が魚になって魚の一生を体験 |
| おみくじ | 魚に関連したおみくじ |
| 水揚げストーリー | その魚がどのように水揚げされたのかを表示 |
| 今が旬！ | 旬の魚を紹介 |
| スタンプラリー | 地域・店・魚のスタンプラリー（ランク・レベル機能付き） |
| 今日の魚 | 日替わりで魚を紹介 |
| 魚種マッチング診断 | MBTI的な魚診断 |
| ファミリーミッション | 家族向けミッション機能 |

### その他

| 機能名 | 説明 |
|--------|------|
| 管理画面（/admin） | 魚登録・魚同士の相性登録を管理 |
| 管理画面の画像登録 | 魚画像をGoogle Photosへアップロードし、画像URL/関連リンクを登録 |
| 魚の相性一覧（一般ユーザー） | トップページで魚同士の相性データを閲覧可能 |
| ローディング画面 | いわしくるくるアニメーション |
| 魚に食べられる演出 | 初めのローディングを魚に食べられるような演出 |
